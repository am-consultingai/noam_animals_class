# חיות העל וגביע העולם 🦁⚽

מצגת אינטראקטיבית (React + Vite) לשיעור על חיות לכיתות ב'.
הנושא: קבוצה מגוונת של חיות מנצחת קבוצה של 11 אריות זהים — לכל חיה כוח-על משלה.
כל התוכן בעברית, בפריסת ימין-לשמאל (RTL).

## הפעלה

```bash
npm install
npm run dev
```

ואז לפתוח את הכתובת שמופיעה בטרמינל (בדרך כלל http://localhost:5173).

## בנייה לגרסת הפצה

```bash
npm run build
npm run preview
```

## ניווט במצגת

- חיצים ⬅️ / ➡️ או מקש רווח — מעבר בין שקופיות
- לחצני "הבא" / "הקודם" בתחתית המסך
- נקודות בראש המסך — קפיצה ישירה לשקופית
- Home / End — לשקופית הראשונה / האחרונה

## עריכת התוכן

כל החיות, כוחות-העל והעובדות נמצאים בקובץ `src/animals.js`.
מבנה השקופיות נמצא ב-`src/slides.jsx`.
