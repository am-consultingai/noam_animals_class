# noam_animals_class 🦁⚽

מצגת אינטראקטיבית (React + Vite) לשיעור על חיות לכיתות ב'.
הנושא: קבוצת חיות מגוונת שמנצחת קבוצה של 11 אריות זהים — לכל חיה כוח-על משלה.
כל התוכן בעברית, בפריסת ימין-לשמאל (RTL).

## צפייה אונליין

האתר מתפרסם אוטומטית ב-GitHub Pages:
**https://am-consultingai.github.io/noam_animals_class/**

## הרצה מקומית

```bash
cd "animals presentation"
npm install
npm run dev
```

## פריסה

כל דחיפה ל-`main` מפעילה את ה-workflow ב-`.github/workflows/deploy.yml`
שבונה את האתר (`animals presentation/`) ומפרסם אותו ל-GitHub Pages.

קוד המקור של המצגת נמצא בתיקייה `animals presentation/`.
